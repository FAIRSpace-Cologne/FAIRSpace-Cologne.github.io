# FAIRSpace Cologne

FAIRSpace Cologne is a project aimed at promoting and facilitating the principles of FAIR (Findable, Accessible, Interoperable, and Reusable) data management in research and academia. This website serves as a hub for articles, publications, and information about the team involved in this initiative.

## Project Structure

The project consists of the following files and directories:

- **index.html**: The homepage of the website, containing the main structure and content.
- **articles.html**: A page dedicated to displaying articles and publications related to the project.
- **team.html**: A page presenting information about the team members involved in the project.
- **style.css**: The stylesheet that defines the layout, colors, fonts, and other visual aspects of the website.
- **script.js**: The JavaScript file that adds interactivity and dynamic behavior to the website.
- **assets/**: A directory containing subdirectories for fonts and icons used throughout the website.
  - **fonts/**: Holds font files used in the website.
  - **icons/**: Contains icon files used in the website's design.
- **.gitignore**: Specifies files and directories to be ignored by Git version control.
- **package.json**: The configuration file for npm, listing the project's dependencies and scripts.

## Setup Instructions

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd FAIRSpace-Cologne.github.io
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `index.html` in your web browser to view the website.
