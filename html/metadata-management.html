Metadata Management

This guide explains how to create, edit, and manage metadata records in FAIRdata Cologne. Learn to add new metadata records, fill in required and optional fields, and maintain quality standards.

Creating a New Metadata Record

Step 1: Access the Metadata Editor

From your dashboard, click on "Create New Record" or navigate to Metadata → New Record in the main menu.

Step 2: Select Metadata Standard

Choose the appropriate metadata standard for your research data:

  • Dublin Core: General purpose metadata standard
  • MIAPPE: For plant phenotyping and genotyping data
  • Custom Standard: For domain-specific metadata requirements

Step 3: Fill in Required Fields

Complete all required fields marked with a red asterisk (*). These typically include:

  • Title
  • Description
  • Creator/Author
  • Date Created
  • Subject/Keywords

Step 4: Add Optional Metadata

Fill in optional fields to improve discoverability and FAIR compliance. Click "Add more fields" to expand the form and see additional metadata options.

Step 5: Save Draft

Click "Save as Draft" to save your progress without publishing. You can return to edit the draft at any time.

Editing Metadata Records

Editing Your Own Records

To edit a record you created:

  1. Go to "My Records" in your dashboard
  2. Find the record you want to edit
  3. Click "Edit" button
  4. Make your changes
  5. Click "Save Draft" or "Update Record"

Version Control

When you update a published record:

  • The system automatically creates a new version
  • Previous versions are archived and accessible
  • You can compare versions side-by-side
  • Changes are tracked with timestamps and usernames

Metadata Field Guide

Title

Provide a clear, descriptive title for your dataset. Keep it concise (under 100 characters) but specific enough to identify the dataset.

Description

Write a comprehensive description of your data. Include:

  • What the data contains
  • How it was collected
  • Any important methodological information
  • Scope and limitations

Keywords

Add relevant keywords separated by commas. Think about terms someone would use to search for your data. Use both specific and general terms.

Related Resources

Link to related publications, projects, or other datasets. Include:

  • DOI or URL
  • Relationship type (e.g., "is cited by", "is related to")
  • Brief description of the relationship

Quality Assurance and Validation

Automatic Validation

As you fill in metadata fields, the system performs automatic validation:

  • Checks for required fields
  • Validates field formats (dates, URLs, etc.)
  • Ensures compliance with selected metadata standard
  • Provides real-time feedback and suggestions

FAIR Assessment

Before publishing, review your FAIR assessment score. This shows how well your metadata aligns with FAIR principles. Aim for at least 80% compliance.

Preview Before Publishing

Always preview your record before publishing to ensure:

  • All information is accurate and complete
  • Formatting is correct
  • Links are functional
  • Sensitive information is not included

Bulk Operations

Bulk Import

To import multiple records at once:

  1. Go to Metadata → Bulk Import
  2. Download the template for your metadata standard
  3. Fill in your metadata in the spreadsheet
  4. Upload the file
  5. Review the import preview
  6. Confirm and process the import

Bulk Edit

To make changes to multiple records:

  1. Select multiple records from the list view
  2. Click "Bulk Edit"
  3. Select which fields to update
  4. Enter new values
  5. Review and confirm changes
