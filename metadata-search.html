Searching Metadata Records

This guide explains how to effectively search for and discover metadata records in FAIRdata Cologne. Learn about search techniques, filters, and advanced query options to find the exact data you need.

Basic Search

Accessing the Search Portal

Navigate to the FAIRdata Cologne homepage and locate the search bar at the top of the page. You can begin searching without logging in.

Simple Search Query

To perform a basic search:

  1. Click on the search box
  2. Type your search terms (e.g., "climate", "RNA sequencing", "agricultural data")
  3. Press Enter or click the search button
  4. Results appear in the search results page

Search Tips

  • Use specific keywords rather than general terms
  • Search is case-insensitive
  • Multiple words are treated as AND (all must match)
  • Partial words will match (e.g., "gene" finds "genes", "genetics", etc.)

Advanced Search

Accessing Advanced Search

From the search results page, click on "Advanced Search" to access more powerful search options.

Search Operators

Use these operators to refine your search:

  • AND: All terms must appear (default, e.g., "climate AND data")
  • OR: At least one term must appear (e.g., "plants OR crops")
  • NOT: Exclude a term (e.g., "data NOT synthetic")
  • Phrases: Search exact phrase in quotes (e.g., "machine learning")
  • Wildcards: Use * for partial matches (e.g., "pheno*" matches phenotype, phenotyping)

Field-Specific Search

Search within specific metadata fields:

  • title: "keyword" - Search only in title
  • creator: "author name" - Search by creator
  • subject: "keyword" - Search by subject/keywords
  • date: "2024" - Search by date

Using Search Filters

Available Filters

Refine your search results using these filters on the left side of the search results page:

  • Data Type: Filter by type of data (e.g., dataset, document, software)
  • Date Range: Filter by publication or creation date
  • Creator/Author: Filter by specific author or institution
  • Subject/Keywords: Filter by predefined keywords
  • Metadata Standard: Filter by metadata standard used (Dublin Core, MIAPPE, etc.)
  • Access Level: Filter by access type (open, restricted, embargoed)
  • FAIR Score: Filter by FAIR compliance score

Applying Multiple Filters

You can combine multiple filters:

  1. Select a filter category
  2. Click on specific filter values to apply them
  3. Results update automatically
  4. Click "X" next to a filter to remove it

Sorting Results

Sort Options

Sort your search results by:

  • Relevance (Default): Results ranked by relevance to your search
  • Date (Newest First): Most recently published records first
  • Date (Oldest First): Oldest records first
  • Title (A-Z): Alphabetical by title
  • FAIR Score: Highest FAIR compliance first
  • Most Viewed: Most popular records first
  • Most Downloaded: Most frequently downloaded records

Viewing Record Details

Accessing Full Record

Click on any record in the search results to view its full metadata. The record detail page shows:

  • Complete metadata information
  • Description and abstract
  • Creator(s) and contributors
  • Keywords and subjects
  • Related resources and links
  • Access and download options
  • Citation information and DOI
  • Usage statistics
  • Version history

Accessing Data

Depending on the record's access level:

  • Open Access: Click "Download" to directly access the data
  • Restricted Access: Click "Request Access" to request data access
  • Embargoed: Shows embargo expiration date; access becomes available after

Saving and Alerts

Save Search

To save a search query for later:

  1. Perform your search with desired filters
  2. Click "Save This Search" button
  3. Give your search a name
  4. Choose to receive notifications for new matching records
  5. Save to your account

Search Alerts

Get notified when new records matching your search criteria are published:

  1. Enable "Email Alerts" when saving a search
  2. Choose alert frequency (daily, weekly, monthly)
  3. Receive email with matching records
  4. Manage all alerts in your profile settings

Exporting Results

Export Search Results

Export your search results in various formats:

  • CSV (spreadsheet format)
  • JSON (machine-readable format)
  • RDF (semantic web format)
  • BibTeX (for citations)

How to Export

To export results:

  1. From search results page, click "Export"
  2. Select desired format
  3. Choose number of results to export
  4. Click "Download"
